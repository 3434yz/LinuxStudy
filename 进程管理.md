## ps -aux


[]()

| 字段    | 含义                                                   |
| ------- | ------------------------------------------------------ |
| USER    | 用户名称                                               |
| PID     | 进程ID                                                 |
| %CPU    | CPU占用百分比                                          |
| %MEM    | 内存占用百分比                                         |
| VSZ     | 虚拟内存占用（kb）                                     |
| RSS     | 内存占用（kb）                                         |
| TTY     | 终端名称                                               |
| STAT    | 状态（S:睡眠，R:运行，Z:僵死进程，T:被跟踪或者被停止） |
| START   | 进程的启动时间                                         |
| TIME    | CPU时间                                                |
| COMMAND | 启动命令                                               |



## ps -ef

全格式的显示

kill [选项] 进程号

-9;强制杀死

killall 进程名称 通过进程名称杀死进程，也支持通配符
